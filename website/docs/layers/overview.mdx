This packages helps managing UI layers. It is primarily used for modals and overlays, but can also come in handy with multi-layer layouts. It supports nested layers.

<PackageBadges package="@weser/layers" />

## Installation

<Callout title="Required dependencies">
  This package relies on [React hooks](https://react.dev/reference/react/hooks)
  and thus requires version 16.3 or higher.
</Callout>

```sh
# npm
npm i --save @weser/layers

# yarn
yarn add @weser/layers

# pnpm
pnpm add @weser/layers
```

## Motivation

Many apps include multiple layers of UI elements. Common examples are modals and overlays.
Managing them in a user-friendly, accessible way is not at all straightforward.

This package is often used in combination with the [useScrollBlocking](/hooks/api/useScrollBlocking) hook.
